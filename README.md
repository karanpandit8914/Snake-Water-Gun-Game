# 🐍 Snake, Water, Gun Game (Python)

## 📌 Overview

**Snake, Water, Gun Game** is a fun and interactive command-line game built using **Python**.
It is similar to the classic **Rock, Paper, Scissors**, but with a twist:

* 🐍 Snake drinks Water → Snake wins
* 💧 Water drowns Gun → Water wins
* 🔫 Gun kills Snake → Gun wins

This project is perfect for beginners who want to understand basic Python concepts like loops, conditionals, and user input.

---

## 🚀 Features

* 🎮 Simple and interactive gameplay
* 🤖 Computer vs Player mode
* 🔁 Multiple rounds support
* 📊 Score tracking system
* ⚡ Lightweight and easy to run

---

## 🛠️ Tech Stack

* **Language:** Python
* **Concepts Used:**

  * Conditional Statements (`if-else`)
  * Loops (`for`)
  * Random Module
  * User Input Handling

---

## 🎯 How to Play

1. Run the Python file
2. Choose one option:

   * `s` for Snake 🐍
   * `w` for Water 💧
   * `g` for Gun 🔫
3. The computer will randomly choose its move
4. The winner is decided based on the rules
5. Scores are updated after each round
